# 🚀 快速开始指南

> 5分钟快速上手AI抖音弹幕游戏主持人

## 📋 准备工作清单

在开始之前，请确保你有：

- [ ] 一台能联网的电脑（Windows/Mac/Linux）
- [ ] Python 3.7+ 已安装
- [ ] 一个阿里云账号（用于申请Qwen API）
- [ ] 你的抖音直播间ID（可选，有模拟模式）

## ⚡ 5分钟快速部署

### 步骤1：下载项目
```bash
# 下载项目代码
git clone https://github.com/your-repo/ai-douyin-host.git
cd ai-douyin-host

# 或者直接下载ZIP文件并解压
```

### 步骤2：安装依赖
```bash
# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate

# 安装依赖包
pip install -r requirements.txt
```

### 步骤3：申请API密钥
1. 访问 [阿里云DashScope](https://dashscope.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 开通DashScope服务
4. 复制API密钥

### 步骤4：配置密钥
```bash
# 复制配置文件
cp .env.example .env

# 编辑.env文件，填入你的API密钥
# DASHSCOPE_API_KEY=your_api_key_here
```

### 步骤5：启动测试
```bash
# 运行测试脚本
python test_api.py

# 如果测试通过，启动主程序
python main.py
```

🎉 **完成！** 你的AI主持人现在已经开始工作了！

## 🔧 基础配置

### 修改AI人格
编辑 `config.yaml` 文件：
```yaml
ai_personality:
  character_type: "活泼少女"  # 可选：活泼少女、幽默段子手、温柔大姐姐、搞怪萌妹
```

### 设置直播间ID
```yaml
douyin_live:
  room_id: "YOUR_ROOM_ID_HERE"  # 填入你的直播间ID
```

### 调整回复频率
```yaml
interaction:
  comment_reply:
    reply_probability: 0.3  # 30%概率回复弹幕，可根据需要调整
```

## 🛟 遇到问题？

### API连接失败
```bash
❌ API调用失败: 401 Unauthorized
```
**解决方案：**
- 检查`.env`文件中的API密钥是否正确
- 确认阿里云账户余额充足
- 验证DashScope服务已开通

### 音频播放失败
```bash
❌ 音频播放失败
```
**解决方案：**
- 确保电脑有音频设备
- Windows用户检查音频驱动
- Linux用户安装音频库：`sudo apt-get install alsa-utils`

### 模块导入失败
```bash
❌ 导入模块失败
```
**解决方案：**
- 确认在项目根目录运行
- 重新安装依赖：`pip install -r requirements.txt`
- 检查Python版本：需要3.7+
